2. Argus QA agents 27 agents

Codex variant: the same 27 Argus agents are available in argus/codex/ as paired *.toml + *.md files, preserving the same bare slugs (odysseus, talos, orion, ...). Claude sonnet source roles map to gpt-5.5 + medium; Claude opus source roles map to gpt-5.5 + xhigh.

Engagement modes (lead Odysseus picks per goal): A full QA audit · B deep bug-hunt (hunters as automatons) · C build tests from scratch (greenfield) · D add to / extend tests in an existing repo (adopt-or-build). Every agent appends a heartbeat every ~10 min to ai_agents_internal/heartbeat/<slug>.log (watch with: tail -f) — to estimate progress.

The Mode (hunt) column = tool hybrid: CLI-first (scripted curl/fetch/node/psql — request/data-level, saves tokens + cache) · browser-MCP (oracle = rendered page; snapshot-frugal). The rest of the roles (core, strategy, automation, triage, report) do no direct hunting — —.
