# solution/findings

Per-finding writeups (one file per confirmed defect / observation) live here. Each
finding cites its oracle (requirement / OpenAPI / WCAG), the reproduction, the
evidence (trace/screenshot/video path), and the linked `BUG-NNN` and regression test.

This directory is part of the Argus QA deliverable contract — keep it in the repo even
when empty (hence this README). The roll-up lives in [`../BUG-LEDGER.md`](../BUG-LEDGER.md);
confirmed bugs also get a RED regression test under `tests/regression/`.
